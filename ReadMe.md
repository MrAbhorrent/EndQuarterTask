# Логика программы
*1* Объявляется пустой массив __tempArray__, который будет содержать результат.

*2* В цикле **for** итерируемся по массиву строк **strings**.

*3* Внутри цикла проверяем, что длина текущей строки string меньше или равна **3**.

*4* Если длина строки меньше или равна **3**, добавляем ее в массив __tempArray__.

*5* Создаем новый массив __resultArray__, в который заносим только не пустые элементы из массива __tempArray__

*6* Возвращаем массив __resultArray__.
